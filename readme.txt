Software optimizado para a versão 3.7 de Python. Para começar verificar primeiro se o 
Python está instalado com comando:

$ which Python


Caso esteja pode-se começar a instalação com o comando: 

$ Python installssre.py 

Isto é um script automatico para instalar: 

- Python3.7
- Pip para python 
- wkhtmltopdf
- imgkit
- pdfkit

A partir daqui apenas se usa a ultima versão de Python (3.7). 
Pode correr a aplicação. 

$ Python3.7 ssre.py
